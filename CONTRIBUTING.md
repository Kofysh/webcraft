# 🤝 Contributing to WebCraft

Thank you for your interest! Any contribution is welcome.

## Getting Started

1. Fork the repo
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/webcraft.git`
3. Install: `npm install`
4. Create a branch: `git checkout -b feat/my-feature`

## Development

```bash
npm run dev   # auto-restart on change
npm test      # run tests
```

## PR Guidelines

- One feature / one fix per PR
- Write a clear description of what you changed and why
- Add a test if you fix a bug
- Update docs if you add a feature

## Code Style

- `const`/`let` only, no `var`
- async/await over raw callbacks
- JSDoc on all exported functions
